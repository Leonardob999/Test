Custom Item Class

Custom item classes are used to make advanced items. For Example your item can check for rightclicking and run an action.

Methods

Use @Override in front of a method to declair that you override a method of the item class

Right Click Behavior

@Override
								public InteractionResultHolder use(Level world, Player player, InteractionHand hand) {
    					return super.use(world, player, hand);
								}

Tool tips

@Override
								public void appendHoverText(ItemStack stack, @Nullable Level worldIn, List tooltip, TooltipFlag flagIn) {
    					tooltip.add(Component.literal("[your tool tip]"));
									
									super.appendHoverText(stack, worldIn, tooltip, flagIn);
									}

Basic Setup

    public class [Your Items name] extends Item {
    	public [Your Items name](Properties pProperties) {
        	super(pProperties); // After pProperties you can add propertiy Settings. For Example pProperties.stacksTo(10);
    	}

   // here you can override methods of the Item Class.
}
