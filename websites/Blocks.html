Basic Blocks

This is for simple Blocks that not need special Abilities like Stone or Dirt.

Strength

Strength controls the hardness of a Block. For Example Stone has a hardness of 1.5 and dirt 0.5. The Block is unbreakable if strenght is -1.0.

Sound

Defines the sound when Block is placed, punched or broken.

Lightlevel

Defines the Light emission of a Block. Can contain a Value of 0 to 15.

Friction

Controlls how slippery a block is. For example ice has a Friction of 0.89.

Advanced Blocks

The Block Properties allows only basic Blocks. If you want more advanced Blocks. For Example if you want your Block to react on Events like Player Interaction. You need an Custom Block class.
